# 😊 Real-Time Facial Expression Recognition using CNN

## 📌 Project Overview

This project detects human facial expressions in real time using a Convolutional Neural Network (CNN) trained on the FER2013 dataset. OpenCV is used to detect faces from the webcam, and the trained CNN predicts the emotion.

---

## 🎯 Features

- Real-time webcam emotion detection
- CNN-based emotion classification
- OpenCV face detection
- Confidence score display
- FER2013 dataset

---

## 📂 Dataset

FER2013

Classes:

- Angry
- Disgust
- Fear
- Happy
- Neutral
- Sad
- Surprise
